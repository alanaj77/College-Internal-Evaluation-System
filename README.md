# 🎓 IEMS - Internal Evaluation Management System
***A modern, full-stack management portal designed to streamline academic administration. This system allows administrators to manage student enrollments, faculty directories, and subject assignments through a sleek, responsive interface.***

## 🚀 Features

- Admin Dashboard: At-a-glance statistics of total students, professors, and subjects.

- Student Management (CRUD): Complete lifecycle management for students including enrollment, profile updates, and record deletion.
  
- Professor Directory: Faculty management with the ability to filter by Designation and Branch.
  
- Live Search & Filtering: High-performance frontend filtering using JavaScript oninput for instant results across directories.
  
- Subject Mapping: Relational database logic to assign specific subjects and branches to faculty members.
  
- Relational Integrity: Backend logic handles child-parent record deletions to maintain database health.

## 🛠️ Tech Stack
### Backend
- Framework: Flask (Python)

- Database: MySQL (Hosted on Aiven Cloud)

- Security: Werkzeug for password hashing and parameterized SQL for injection prevention.

### Frontend
- Templating: Jinja2

- Styling: Modern CSS3 (Dark/Light Mode compatible)

- Interactivity: Vanilla JavaScript (ES6+)
